Development

WebThings Gateway is an open source project maintained by a community of volunteers.

To get started hacking on the gateway, see the README and wiki.

The API exposed by the gateway is called the Web Thing API. We are currently in the process of migrating this API to conform with W3C Web of Things standards. You can see an overview of differences between the two here.
